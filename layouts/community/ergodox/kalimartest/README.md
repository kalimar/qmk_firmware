## Layout

### Keymap 0: Basic layer
```
,--------------------------------------------------.           ,--------------------------------------------------.
|   `    |   1  |   2  |   3  |   4  |   5  |  6   |           |  7   |   8  |   9  |   0  |   -  |   =  |Keypad  |
|--------+------+------+------+------+-------------|           |------+------+------+------+------+------+--------|
| TAB    |   Q  |   W  |   E  |   R  |   T  | TOG  |           | TOG  |   Y  |   U  |   I  |   O  |   P  |   \    |
|--------+------+------+------+------+------|  L1  |           |  L1  |------+------+------+------+------+--------|
|Esc/ctrl|   A  |   S  |   D  |   F  |   G  |------|           |------|   H  |   J  |   K  |   L  |   ;  |   '    |
|--------+------+------+------+------+------|Hyper |           | Meh  |------+------+------+------+------+--------|
|(/LShift|  Z   |   X  |   C  |   V  |   B  |      |           |      |   N  |   M  |   ,  |   .  |   /  |)/RShift|
`--------+------+------+------+------+-------------'           `-------------+------+------+------+------+--------'
 |invtnum| mouse|      | LGUI | CMD  |                                       | CMD  | RGUI |  [   |   ]  |invtnum|
  `----------------------------------'                                       `----------------------------------'
                                       ,-------------.       ,---------------.
                                       | Dim  |Bright|       | Vol D| Vol U  |
                                ,------|------|------|       |------+--------+------.
                                |      |      |      |       | mute |        |      |
                                | Space| Bksp |------|       |------|  Ent   |Space |
                                |      |      |      |       |      |        |      |
                                `--------------------'       `----------------------'
```
* Hyper   - holding Alt+Shift+Ctrl+Cmd
* Meh     - holding Alt+Shift+Ctrl
* Keypad  - right layer 2 lock
* mouse   - left layer 2 lock
* invtnum - layer 3 lock
* ------  - transparent key
